What is HTML?

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

What is HTML used for?

HTML used to create web pages.

Where did HTML come from?

Tim Berners-Lee, an English computer scientist, best known as the inventor of the World Wide Web developed HTML at CERN, a European research organization that operates the largest particle physics laboratory in the world, using SGML.

How is HTML written?

HTML is written using tags and code.Tags are the hidden keywords within a web page that define how the browser must format and display the content. Code is the coding language that is used to display web pages.

What is CSS?

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language.

How do tags work?

Tags basically let you do things like make headings or type like this. They are basically the building blocks of css.

This is how you write a tag

h8{font-size:35px;}

What is an attribute?

Attributes provide additional information about an element

How do you change attributes?

To change an attribute you have to delete the code that you wrote with the attribute and then write code with a new attribute

Examples of attributes

() Defines a comment
!DOCTYPE Defines the document type
